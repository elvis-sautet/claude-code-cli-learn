# 📹 Camera & Meetings

Open your webcam and launch **free** video meetings — no account, no backend, no cost.
Built with plain HTML + CSS, **type-safe TypeScript**, [Iconify](https://iconify.design) icons,
and covered by **Playwright E2E tests**.

Meetings use the free [Jitsi Meet](https://meet.jit.si) embed API.

## Features

**Camera**
- Open / stop the webcam with a live, mirrored preview
- Pick which camera and microphone to use (device settings ⚙️)
- Toggle camera and mic on/off from a floating control bar

**Meeting**
- Launch / join a real, shareable Jitsi room
- Generate a random room name 🎲
- Copy the invite link to the clipboard
- In-call controls: mute, stop video, share screen, copy link, leave —
  all synced to the live Jitsi state

## Run it

You need [Node.js](https://nodejs.org) installed.

```bash
npm install        # installs TypeScript + Playwright
npm start          # compiles app.ts and serves at http://localhost:5173
```

Other scripts:

```bash
npm run build      # compile app.ts -> app.js once
npm run watch      # recompile on every save
npm run serve      # static server only (no compile)
```

Then open `http://localhost:5173`, click **Open camera**, allow the permission prompt,
enter a room name, and click **Launch / Join**.

> ⚠️ The camera only works over `http://localhost` or `https://`. Opening `index.html`
> as a `file://` path will block camera access — always use the dev server.

## Tests

End-to-end tests run in headless Chromium with a fake camera/mic, and stub the Jitsi
API so they're fast and offline.

```bash
npm test           # compile, then run the Playwright suite
npm run test:ui    # interactive Playwright UI mode
```

Covered flows: initial UI state, camera open/toggle/stop, room-name slugifying,
copy-link to clipboard, and the full meeting launch → toggle controls → leave cycle.

## Files

| File                 | Purpose                                              |
| -------------------- | ---------------------------------------------------- |
| `index.html`         | Markup; loads Iconify, the Jitsi API, and `app.js`   |
| `styles.css`         | Styling                                              |
| `app.ts`             | Type-safe camera + meeting logic (source)            |
| `app.js`             | Generated by `tsc` — do not edit by hand             |
| `serve.mjs`          | Minimal static dev/test server                       |
| `tsconfig.json`      | Strict TypeScript config                             |
| `playwright.config.ts` | E2E config (fake media devices, auto-served app)   |
| `tests/app.spec.ts`  | E2E test suite                                       |
